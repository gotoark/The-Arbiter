# The Arbiter - Cyberpunk Tech Theme

## Visual Design System

### Color Palette
- **Primary Neon**: Cyan (#00ffff) - Main UI elements, borders, text
- **Secondary Neon**: Pink/Magenta (#ff00ff) - Accents, highlights
- **Tertiary Neon**: Green (#00ff00) - Success states, advantages
- **Warning Neon**: Orange (#ff6600) - Trade-offs, cautions
- **Error Neon**: Red (#ff0000) - Errors, vulnerabilities
- **Info Neon**: Yellow (#ffff00) - Recommendations, verdicts

### Typography
- **Headers**: Orbitron (futuristic, geometric)
- **Body**: Rajdhani (clean, technical)
- **Code/Data**: Monospace for technical content

### Effects
- **Neon Glow**: Text shadows and box shadows for cyberpunk feel
- **Glitch Effect**: Animated title with color displacement
- **Scan Lines**: Subtle overlay for retro CRT monitor effect
- **Grid Background**: Subtle cyan grid pattern
- **Animated Elements**: Pulsing indicators, sliding gradients

### UI Components
- **Cyber Cards**: Dark backgrounds with neon borders and glow
- **Neon Buttons**: Glowing borders with hover effects
- **Score Bars**: Gradient-filled progress bars with glow
- **Input Fields**: Dark with neon borders and focus effects

### Language Style
- **Uppercase Labels**: Technical, military-style labeling
- **System Messages**: Computer terminal-style communication
- **Error Messages**: Dramatic, system failure language
- **Status Indicators**: Online/offline system status displays
